1000으로 나누어지지 않으면서  100으로 나누어지면  평년이고,

100으로 나누어지지 않으면서 4로 나누어지면 윤년이고,

4로 나누어지지않으면 평년이다. 어느 년도인지 입력하면 윤년인지 평년인지 알려 주는 프로그램을 작성하라. 단, 영이나 음수가 입력되면 종료된다.(leap.c)
